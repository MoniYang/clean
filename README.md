## commit 格式
 - [Core / Credit / Lic][View / Site / Project / Feature] Message
 - 以上兩個標籤或三個標籤可視情況搭配
 - 舉例:
    - [Core][Asia] 調整 odds table...
    - [Core][Asia/China] 調整 xxx...
    - [Core][PopupPage] 調整 xxx...
    - [Credit][IBCBet] 客製顏色 xxx...
    - [Lic][Asia] 同步資源檔案...
    - [Lic][188] 新增 xxx...
    - [Lic][China] 12Bet 客製顏色 xxx...
